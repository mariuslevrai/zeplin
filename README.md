# Zeplin
Zeplin is only available on linux 🐧
# Requirements
- x64 or x32 cpu
- Python 3.12.3
- Virtual environments to execute certains lib
- git 2.43.0 or less
# Installation
- Install git with apt :
```bash
sudo apt-get install git
```
- Install zeplin 👨‍💻 :
```bash
git clone https://github.com/mariuslevrai/zeplin
```
- Run the app 🖥️ :
```bash
bash run.sh
```
